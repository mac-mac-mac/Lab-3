Note that in Lab 6, the menu items will have to display inline instead of block, so they can sit side-by-side. No bullets or underlines on the menu items. Make them look like buttons with CSS. Use semantic tags as shown. So that every part of the layout is clearly visible, please add a different background color to each. Please create attractive styles for the text items. Add three buttons to the article section. When pressed, each button should turn the background color of just the article section another color. Label each of the buttons with the color that it changes the background color to. You will need to use Javascript to accomplish this.
